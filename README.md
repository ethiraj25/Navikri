# Navikri — AI-Powered Fresher Hiring Automation

> Built during internship at **Innovacx, Hyderabad** · Python · Flask · Claude API · June 2025

---

## What is Navikri?

Navikri is a full-stack AI-powered recruitment platform designed to automate the end-to-end fresher hiring process — from job description setup to final call letters — with a human HR override at every stage.

**The problem it solves:** A typical HR team manually reads hundreds of resumes, writes MCQ papers, evaluates candidates, and sends individual call letters. Navikri automates all of it in minutes.

---

## The 7-Phase Pipeline

```
JD Setup → Resume Screening → HR Review → MCQ Generation → Online Assessment → Final Review → Call Letters
```

| Phase | What happens |
|---|---|
| 1. JD Setup | HR defines the job description and required skills |
| 2. Resume Screening | AI reads, parses, scores, and ranks all uploaded resumes |
| 3. HR Review | HR reviews AI shortlist — can override any decision |
| 4. MCQ Generation | AI auto-generates a custom question paper based on the JD |
| 5. Online Assessment | Shortlisted candidates take the test on the platform |
| 6. Final Review | AI scores tests, HR makes final selection |
| 7. Call Letters | Personalised call letters auto-drafted and sent via email |

## Features

- **Resume Parser** — extracts skills, experience, and education from any PDF resume
- **AI Scoring Engine** — scores each candidate against the JD requirements
- **MCQ Generator** — builds a custom test paper from the job description automatically
- **Candidate Summariser** — generates a one-line summary of each candidate for HR
- **Call Letter Drafter** — writes personalised call letters for selected candidates
- **HR Override** — HR retains full control and override at every single stage
- **Dark-themed SPA** — clean, modern single-page interface
